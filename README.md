# Ballet Tribute Page

This project was completed during the PERSEVERE Coding Bootcamp while I was incarcerated.

The assignment was to create a tribute page using HTML and CSS. I chose ballet as the subject because it has been a lifelong passion and an important part of my personal journey. During my incarceration, I founded and taught the **Ballet Behind Bars** program, making this project especially meaningful to me.

The project is preserved in its original form to document my early growth as a front-end developer.
